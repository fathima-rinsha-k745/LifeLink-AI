# 🩸 LifeLink AI
[![Django Tests](https://github.com/fathima-rinsha-k745/LifeLink-AI/actions/workflows/django.yml/badge.svg)](https://github.com/fathima-rinsha-k745/LifeLink-AI/actions/workflows/django.yml))

## Problem

Finding blood donors during emergencies can be difficult and time-consuming. LifeLink AI helps connect blood donors and recipients quickly through intelligent donor matching and emergency response features.

## Planned Features

* User Registration & Login
* Donor Profile Management
* Blood Request Creation
* AI-Based Donor Matching
* Emergency Notification Generator
* Swagger API Documentation

## Tech Stack

* Python
* Django REST Framework
* Supabase PostgreSQL
* OpenAI/Claude API
* Railway Deployment

## Project Status

Currently in MVP Development Phase.
